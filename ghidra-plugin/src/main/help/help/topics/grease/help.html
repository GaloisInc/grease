GREASE Analysis Pass Help

The GREASE annotation pass runs as a Ghidra function pass, running on all functions starting at callgraph leaves. General information about GREASE can be found in the GREASE documentation To run the analysis one can either use the auto-analysis plugin and enable the Grease Analyzer or select Analysis->One Shot->Grease Analyzer.

The Grease analyzer has several options that can be configured:

  • Raw mode: loads the binary at a fixed offset as a non-ELF, useful for firmware. By default loads at the image base.
  • Load base: Changes the base that the binary is loaded at in raw mode.
  • Timeout: The timeout in milliseconds for a single function

The analysis can also be run as a one shot analysis via the one shot menu.
